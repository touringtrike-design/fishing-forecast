## 🎨 ДИЗАЙН КОНЦЕПЦІЯ

### Стиль:

- **Мінімалістичний** - фокус на функціональності
- **Природна палітра** - сині/зелені тони (вода, природа)
- **Чіткі акценти** - червоний/помаранчевий для важливого
- **Читабельність** - великі шрифти, контрастні кольори

### Кольорова схема:

css

````css
/* Основні кольори */
--primary-blue: #2196F3;      /* Вода */
--primary-green: #4CAF50;     /* Природа */
--accent-orange: #FF9800;     /* Акценти */
--accent-red: #F44336;        /* Попередження */

/* Фон */
--bg-dark: #1A1A2E;          /* Темний режим */
--bg-light: #F5F5F5;         /* Світлий режим */
--surface: #FFFFFF;

/* Текст */
--text-primary: #212121;
--text-secondary: #757575;
--text-disabled: #BDBDBD;

/* Градієнти */
--gradient-water: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
--gradient-success: linear-gradient(135deg, #11998e 0%, #38ef7d 100%);
```

#### Деталі головного екрану:

**Header (верхня панель):**
```
┌─────────────────────────────────────────────────┐
│ 🎣 Прогноз клювання    🔍    ☰ Menu  👤 Profile │
│                                                 │
│ 📍 Київ, Україна                    23°C ☀️     │
└─────────────────────────────────────────────────┘
```

**Карта (інтерактивна):**
- **Водойми** - сині полігони (з OSM)
- **Гарячі точки** - кольорові маркери:
  - 🔴 Червоний: Відмінно (80-100%)
  - 🟠 Помаранчевий: Добре (60-80%)
  - 🟡 Жовтий: Середньо (40-60%)
  - 🔵 Синій: Погано (< 40%)
- **Вибрана точка** - великий червоний маркер 📍
- **Контроли:**
  - Zoom +/-
  - Компас 🧭
  - Моя локація 📍
  - Пошук 🔍

**Прогноз панель (знизу):**
- **Висувається** з низу екрану
- **3 стани:**
  - Згорнута: тільки рейтинг ⭐⭐⭐⭐⭐
  - Середня: погода + наживки
  - Повна: детальний прогноз

## 📱 RESPONSIVE DESIGN

### Mobile (< 768px):
- Панель прогнозу - **повноекранний modal**
- Карта - **на весь екран**
- Меню - **burger menu**
- Карточки - **стак вертикально**

- Панель прогнозу - **slide-up drawer**
- Карта - **70% екрану**
- Меню - **бокова панель**

### Desktop (> 1024px):
- Панель прогнозу - **fixed справа (30%)**
- Карта - **ліворуч (70%)**
````
